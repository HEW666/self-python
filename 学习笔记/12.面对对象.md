# 成员方法的定于语法
- 这就是一个类的方法和使用
```python
class User:  
    name = None  
    age = None  
  
    def say_hi(self):  
        print(f"Hi大家好,我是{self.name},今年{self.age}岁.")  
  
  
use = User()  
use.name = "张三"  
use.age = 28  
use.say_hi()
```
- 类中定义的属性(变量),称之为 **成员变量**
- 类中定义的行为(函数),称之为 **成员方法**
- self 关键字是成员方法定义的时候,必须填写的
- 在方法内部,想要访问类的成员变量,必须使用self
