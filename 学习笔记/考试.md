# 函数
> 返回一个包含商和余数的元组 (a // b, a % b)
```python
divmod(a, b)
```
> 返回x的y次方
```python
 pow(x, y)
```
> 返回四舍五入 n是保留小数位数
```python
round(x [, n])
```
> 取最大值 和 取 最小值
```python
max(x, y, z, ...)
min(x, y, z, ...)
```
---
> python格式化输出
```python
print('{a} {b} {a}'.format(a='千锋',b='教育'))
千锋 教育 千锋
```
## 处理函数
> 返回值是当前整数对应的 ASCII 字符。i是10进制也可以是16进制 256以内(0~255)
```python
chr(i)
```

```python
ord('a')   # 返回值是对应的十进制整数
oct(10)    # 返回 8 进制字符串。
hex(x)     # 返回16进制数，以字符串形式表示。
```
