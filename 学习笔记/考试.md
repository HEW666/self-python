# 函数
> 返回一个包含商和余数的元组 (a // b, a % b)
```python
divmod(a, b)
```
> 返回x的y次方
```python
 pow(x, y)
```
> 返回四舍五入 n是保留小数位数
```python
round(x [, n])
```
> 取最大值 和 取 最小值
```python
max(x, y, z, ...)
min(x, y, z, ...)
```

> python格式化输出
```python
print('{a} {b} {a}'.format(a='千锋',b='教育'))
千锋 教育 千锋
```
## 处理函数
> 返回值是当前整数对应的 ASCII 字符。i是10进制也可以是16进制 256以内(0~255)
```python
chr(i)
```

```python
ord('a')   # 返回值是对应的十进制整数
oct(10)    # 返回 8 进制字符串。
hex(x)     # 返回16进制数，以字符串形式表示。
```
# 程序的控制结构
## 程序的三种控制结构  
顺序结构、分支结构、循环结构  
## 程序的分支结构  
单分支结构(if)、二分支结构(if else)、多分支结构(if elif else)  
## 程序的循环结构  
遍历循环(for)、无限循环(while)、跳出循环(break) 跳过本轮循环(continue)  
## 程序的异常处理：
try-except
