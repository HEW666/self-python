# 列表的下标（索引）
列表中的每一个元素，都有其位置下标索引，
**从前向后的方向，从0开始，依次递增**
```python
data = ["0", "1", "3"]
print(data[0])
```
- 使用 print(data[0]) 进行索引
**从后往前的方向，从-1开始，依次递增**
```python
data = ["-3", "-2", "-1"]
print(data[-2])
```
# 嵌套列表的索引
**索引 yellow 单词**
```python
data = [["home", "hello"], ["yellow", "black"]]
print(data[1][0])
```
